Консольное приложения для учета автобусов в парке.
Для использования программы необходимо наличие Python (3й - версии)
Для запуска проекта необходимо:
    - в терминале перейти в директорию проекта (BusPark_PyQT5)
    - запустить файл databases.py (>python3 databases.py)
    - выполнять работу следуя инструкциям МЕНЮ
Возможны следующие функции:
    - создание базы данных (по умолчанию создана тестовая база test_db_1)
    - добавление транспортного средства (ТС) в базу данных (БД)
    - удаление ТС из БД
    - поиск по номеру ТС в БД
    - просмотр сведений по всем ТС
    - просмотр сведений по циклам (на линии, в парке, на ТО)
    - добавление ТС в соответсвующий список в зависимости от цикла работы (на линии, в парке, на ТО)
      с фиксацией времени изменения.

Для удаления БД необходимо удалить файл БД (в данном случае test_db_1).
Создание своей (новой) БД:
    - открыть файл databases.py (любым текстовым редактором)
    - изменить аргумент функции 'test_db_1' (предпоследняя строка) conn = BusParkStorage('test_db_1') на любое свое имя.
    - далее выполнить запуск программы (см выше)
    - заполнить БД своими данными
